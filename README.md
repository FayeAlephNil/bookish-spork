# bookish-spork

This repository contains the code for a project concerning the spatial models of elections, group inefficiency, and districted elections. Particularly, the project aims to explore the interactions between these three aspects of social choice theory. 

The primary way to run the code which produces the data for our project is to run
```python
python main.py
```
provided that you have votekit and its dependencies installed. This will prompt you for which election type to run and whether you want the results to be displayed in a viewer or not. Either way, the results will be saved to the images/ folder. Note that the main.py program uses seeded runs to ensure reproducibility.
